常用 Markdown 語法

發表於 2020-03-14 更新於 2020-03-16 分類於 筆記

因為還不太熟悉 Markdown 語法，因此把一些可能會用到語法記錄在這裡，忘記的時候可以來這裡翻。

標題

清單

分為有序清單及無序清單兩種：

有序清單

在行首加入數字及一個點

範例：

  1. 第一條
  2. 第二條

無序清單

無序清單有星號、加號、減號等方法，但效果相同，分別為在行首加入*、+、-

範例：

  • 星號
    • 第二階

程式碼

分為小區塊標記跟大區塊標記

小區塊

使用`包住要標記的區塊

範例：
要包住的內容在後面test()

大區塊

大區塊支援高亮程式碼的效果，使用三個`包住要高亮的程式碼區塊，並在前面的`後加上使用的語言

連結

使用[]及()來標記，分為兩個部分[畫面上顯示的內容] (連結的網址)

範例：
點我去 Google

圖片

使用!接著[Alt 文字]接著(圖片路徑)
