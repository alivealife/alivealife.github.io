JavaScript 學習紀錄

  1. html 在渲染網頁時由上而下，逐行渲染，故 JS 載入的位置須注意是否要放在最後面，以確保載入時不會出錯。

  2. JavaScript 是屬於單執行緒的程式語言，程式碼遇錯後會中斷不再繼續執行。

  3. 可以使用 typeof( ) 來獲取資料的型態。

  4. 字串的數字相加會得到錯誤的結果(例：10 + 5 = 105)，但是相乘會正確(例：10 * 5 = 50)。
